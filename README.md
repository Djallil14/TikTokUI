# TikTokUI

### Xcode 14
### iOS 16
TikTok Home page UI mainly made in SwiftUI, only the player is made with UIKit/AVKit because we can't disable controls on SwiftUI VideoPlayer and looping is less cleaner than using the AVQueue Player
